AXSizzle is a visually rich, premium-style web platform for ordering custom artwork. Designed for simplicity and elegance, this site offers:

🎨 A clean, interactive homepage with aesthetic blur effects and smooth UI

🛒 An Order Art section for commissioning personalized pieces

💬 A Live Query feature for real-time interaction or inquiries

👥 A stylish Meet Us page to connect with the creators

📄 Terms & Conditions to build trust and transparency

🔐 A modern, blurred Login/Sign Up popup for user access

🎥 A full-screen ambient video background for immersive experience

All built with responsive HTML, CSS, and JavaScript — perfect for showcasing artistic projects in a professional, user-focused way.
